GPT-5.3-Codex 连续运行约 25 小时不间断，使用约 1300 万 token，生成约 3 万行代码，从零构建了一个设计工具。

The real shift is time horizon

实质性的变化是：Agent 可以在更长时间内保持连贯性，端到端完成更大块的工作，并在不丢失线索的情况下从错误中恢复。

Agent 循环：计划 → 编辑代码 → 运行工具 → 观察结果 → 修复故障 → 更新文档/状态 → 重复

The key idea: durable project memory

支撑长任务运行的核心是四份持久化文档：

  • Prompt.md：规格 + 交付物（冻结目标）
  • Plan.md：里程碑 + 验证（将开放性工作转化为检查点）
  • Implement.md：执行指令（操作手册）
  • Documentation.md：状态 + 决策（共享记忆和审计日志）

What the agent built

这是 Agent 从零构建的设计工具功能列表：

  1. 画布编辑（框架、组、形状、文本、图像、按钮、图表）
  2. 实时协作
  3. Inspector 控件
  4. 图层管理
  5. 参考线/对齐/吸附
  6. 历史快照 + 恢复
  7. 回放时间线 + 分支
  8. 原型模式
  9. 评论
  10. 导出

Takeaways

让长任务可靠运行的关键要素：

  • 清晰的目标（Prompt.md 冻结规格）
  • 检查点里程碑（Plan.md 将大任务分解为可验证步骤）
  • 操作手册（Implement.md 提供明确的执行指令）
  • 持续验证（运行工具检查结果）
  • 实时状态/审计日志（Documentation.md 记录决策和进度）

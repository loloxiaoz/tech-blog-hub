模型 API 更新

上周我们的模型发布了新的更新，改善了 Tool Use 的使用体验，欢迎开发者体验。
Kimi 大模型允许用户在对话中指定外部工具
 (opens in a new tab)
，来完成特定的任务，Kimi 会根据对话的上下文，智能地判断应该调用的外部工具，以扩展自己的能力，满足用户的需求。我们观察到大部分用户更希望模型直接调用工具，而不返回思考过程，目前已经默认模型使用这一行为，可以让模型的回复更加简洁，节省 Tokens。同时，用户也可以通过 Prompt 调整让模型返回具体的思考过程，开发者可以根据具体业务场景自行选择。

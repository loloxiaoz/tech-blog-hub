Qwen3 技术概览：混合推理与多尺寸开源

2025-04-29约 1 分钟Qwen开源模型混合推理
阅读原文
中文摘要
Qwen3 发布多个尺寸的开源模型，引入"思考/非思考"双模式，并在多语言与智能体能力上显著增强。

Qwen3 是阿里通义千问系列的新一代开源大模型，覆盖从稠密（Dense）到混合专家（MoE）的多种尺寸，强调统一的混合推理能力与开源生态。

核心特性

  • 混合推理双模式：同一模型支持"思考模式（thinking）"与"非思考模式（non-thinking）"，可按任务难度切换，平衡质量与延迟。
  • 多尺寸覆盖：从适合端侧的小模型到旗舰 MoE，满足不同部署预算。
  • 多语言增强：支持上百种语言与方言。
  • 更强的智能体能力：原生支持工具调用与 MCP 风格的接口。

思考模式 vs 非思考模式

模式适用特点
思考模式数学、代码、复杂推理先生成推理链再作答，质量高、耗时长
非思考模式闲聊、简单问答直接作答，低延迟

开发者可以通过参数显式控制是否开启思考模式：

messages = [{"role": "user", "content": "证明根号2是无理数"}]
# enable_thinking=True 时模型会先输出推理过程
out = model.generate(messages, enable_thinking=True)

开源与生态

Qwen3 以宽松许可证开源，模型权重同时发布在 ModelScope 与 HuggingFace。社区可基于其做微调、量化与部署。

国内官方技术博客中，Qwen 的 GitHub Pages 站点是少数结构规范、中英双语且带 RSS 的范例。

要点总结

Qwen3 用"一个模型、两种推理模式"的设计，把推理质量与响应速度的取舍交给使用者，同时延续了 Qwen 系列一贯的全尺寸开源策略。
